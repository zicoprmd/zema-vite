# Timeline Photos

Simpan foto-foto perkembangan anak Anda di folder ini.

## Naming Convention
- `6bulan.jpg` - Usia 6 bulan
- `1tahun.jpg` - Usia 1 tahun
- `1_5tahun.jpg` - Usia 1.5 tahun
- `2tahun.jpg` - Usia 2 tahun
- `2_5tahun.jpg` - Usia 2.5 tahun
- `3tahun.jpg` - Usia 3 tahun
- `3_5tahun.jpg` - Usia 3.5 tahun
- `4tahun.jpg` - Usia 4 tahun
- `4_5tahun.jpg` - Usia 4.5 tahun
- `5tahun.jpg` - Usia 5 tahun

## Format
- Didukung: .jpg, .jpeg, .png, .webp
- Disarankan: .jpg dengan resolusi yang baik
